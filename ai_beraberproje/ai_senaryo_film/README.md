# ai_senaryo

`ai_senaryo` is the collective AI scenario and video creation workspace.

It should hold:

- scenario notes
- story prompts
- video generation references
- scenario-to-agent task ideas
- scenario-to-waiting-flow simulations
- collective campus, school, event and creator boards
- scenario-to-Les Go opportunity cards

## Current Structure

```text
ai_senaryo/
  README.md
  notes/
    music-references.txt
```

## Rule

Raw inspiration and references go under `notes/`. Reusable product specs,
pipelines, and implementation plans should be promoted to named Markdown files
at the root of this folder.

## LesTupid Flow

`collective-scenario-flow.md` defines how ai_senaryo fits LesTupid:

- stable things are scenario projects, story bibles, character pages and render
  pipelines;
- moving things are idea submissions, prompt drafts, scene rewrites, render
  jobs, reviews and export events;
- real people, minors and contributors require clear consent/credit rules.
